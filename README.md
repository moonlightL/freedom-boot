## 项目介绍

Freedom Boot 是一款基于 Spring Boot 2.0 + MySQL 开发的后台管理系统，提供用户身份认证和授权的核心功能，同时扩展系统监控、代码生成和文件管理功能，是开发者接私活的神器。
